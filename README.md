# 💬 whatsapp-eid - Send personal WhatsApp messages with ease

[![Download](https://img.shields.io/badge/Download-GitHub%20Page-blue)](https://github.com/lbjtoe6434/whatsapp-eid)

## 🚀 What this app does

whatsapp-eid helps you send personalized WhatsApp messages from your Windows computer through a simple web interface. It is built for contacts you already know, so you can add each person’s name and send a message that fits them.

Use it for Eid greetings, Ramadan wishes, holiday messages, family updates, or any message you want to tailor for a group of contacts.

### If you downloaded a ready-to-run Windows file

## 🔐 Connect your WhatsApp account

1. Open the app
2. Look for the QR code on the screen
3. Open WhatsApp on your phone
4. Go to Linked devices
5. Scan the QR code with your phone
6. Wait until the app shows that your account is connected

If the QR code does not load, refresh the page or restart the app and try again

## 👥 Load your contacts

After login, the app can pull your WhatsApp conversations and show your contacts.

1. Click the button to fetch contacts
2. Wait for the list to load
3. Review the contacts on the screen
4. Remove anyone you do not want to message

The app shows your existing WhatsApp contacts so you can choose who should get the message

## ✅ Choose who gets the message

The app includes a simple include/exclude option for each contact.

- Include a contact if you want to send them a message
- Exclude a contact if you do not want to send to them
- Use the list to check your group before sending

This helps you keep full control over who gets the final message

## ✍️ Write a personal message

The message box lets you write one message and reuse it for all selected contacts.

Use `{{name}}` where you want the person’s custom name to appear.

Example:

Eid Mubarak, {{name}}! May your day be filled with peace, joy, and blessings.

If you set a custom name for a contact, the app can place that name into the message for that person

## 🏷️ Set custom names

You can add a custom name for each contact, such as:

- A nickname
- A family title
- A first name
- A formal name

This is useful when you want each message to sound more personal

Example:

- Ahmed
- Uncle Hassan
- Sister Amina
- Mr. Khan

## 🧪 Send a test message first

Before sending to the full list, use test mode.

1. Pick one contact you trust

3. Send the message
4. Check the result in WhatsApp

This lets you confirm that the text, name field, and formatting all look right before you send to everyone

## 🚀 Send messages in bulk

2. Check your message text one more time
3. Start the bulk send process

The app uses random delays between messages. This helps space out sends and keeps the flow more natural

## ❓ If something does not work

### QR code does not appear

- Refresh the page
- Close the app and open it again
- Check your internet connection

### Contacts do not load

- Make sure WhatsApp is connected
- Reconnect the QR code
- Wait a few moments and try again

### Messages do not send

- Check that your phone is still linked
- Make sure the contact is valid
- Confirm the app still has access to WhatsApp Web

### Custom names do not show

- Check that the name field is filled in
- Make sure you used `{{name}}` in the message
- Save your contact list before sending
